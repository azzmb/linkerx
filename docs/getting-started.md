# 简单介绍

LinkerX 是一个基于 gost 的流量转发面板。

就是一个用来管理 [gost](https://gost.run/) 转发工具的面板

- 基于 gost 转发工具
- 转发链、端口转发
- 用户管理
- 流量统计
- 服务器监控
- 使用需要作者授权，不建议在生产环境使用
